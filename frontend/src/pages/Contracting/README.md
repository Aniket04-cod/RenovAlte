# Contracting Module

## Overview

The Contracting module allows users to manage contractors and the contracting process for renovation projects. It provides a step-by-step workflow for matching, inviting, comparing, and selecting contractors.

## Features

- **Matching**: Find and filter contractors based on project type, location, and rating
- **Invite**: Send invitations to selected contractors
- **Offers**: Review contractor offers
- **Compare**: Compare multiple contractor offers side-by-side
- **Draft**: Create and review contract drafts
- **Sign**: Finalize contracts with selected contractors

## Usage

The Contracting module is accessible through the main navigation when a project is selected. It automatically filters contractors based on the selected project's type and location.

## Workflow Steps

1. **Matching** - Browse and select contractors that match your project requirements
2. **Invite** - Send invitations to selected contractors
3. **Offers** - Review received offers from contractors
4. **Compare** - Compare offers to make informed decisions
5. **Draft** - Create contract drafts
6. **Sign** - Finalize and sign contracts
